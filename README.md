# Academic Website Template

A minimal, clean, and responsive academic website template for PhD students and researchers.

## Structure

- **index.html**: Main content file. Edit this to add your information.
- **styles.css**: Styling for the website. Modify colors and fonts here.
- **assets/**: Directory for images. Place your profile picture here.

## How to Customize

1.  **Personal Info**: Open `index.html` and replace "Your Name", "Your University", etc., with your details.
2.  **Profile Image**: Replace `assets/profile-placeholder.svg` with your own photo (e.g., `profile.jpg`) and update the `src` attribute in line ~48 of `index.html`.
3.  **Links**: Update links to your CV, Google Scholar, and social media profiles.
4.  **Content**: Add your research projects, publications, and teaching experience in the respective sections.

## Hosting

Since this repository is named `username.github.io`, it is ready to be hosted on GitHub Pages. Simply push these files to your GitHub repository, and your site will be live at `https://username.github.io`.
